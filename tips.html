Before You Start

  • Save your work (documents, browser forms, anything important).
  • Restart first. It fixes a surprising number of issues.
  • One change at a time. Try a step, then check if it helped.
  • If something looks suspicious (popups asking for money or passwords), stop and see the Scams page.

Speed Up a Slow Windows PC (Quick Checklist)

  1. Restart the computer. Then test performance again.
  2. Close unused programs and browser tabs. Too many tabs can slow things down.
  3. Check free storage space. Aim for at least 10–20% free space on the main drive.
  4. Uninstall apps you don’t use. Settings → Apps → Installed apps.
  5. Disable heavy startup apps. Task Manager → Startup apps → disable what you don’t need.
  6. Run Windows Update. Settings → Windows Update → Check for updates.
  7. Run a malware scan. Windows Security → Virus & threat protection → Quick scan.

If your computer is still slow after these steps, it may need deeper troubleshooting (storage health, RAM usage, background services, or hardware upgrades).

Internet or Wi-Fi Not Working

  1. Check other devices. If your phone also has no internet, it may be your router/ISP.
  2. Toggle Wi-Fi off and on. Then reconnect to the correct network.
  3. Restart your router. Unplug it for 20 seconds, plug back in, wait 2–3 minutes.
  4. Forget and reconnect. Remove the Wi-Fi network and reconnect with the password.
  5. Try a different website. Sometimes one site is down, not your internet.

If only one device has the issue, it can be a driver problem, DNS issue, or a device setting.

Printer Not Printing

  1. Check power and connections. Is it on? Is the USB/Ethernet cable snug?
  2. Confirm the right printer. Make sure you’re printing to the correct printer.
  3. Check the print queue. Clear stuck jobs and retry.
  4. Restart the printer. Power off/on, then try a test print.
  5. Restart the computer. Then try again.

If you recently changed Wi-Fi or got a new router, the printer may need to be reconnected to the network.

Device Basics (Plain Language)

  • Storage is your long-term space (files, apps). Low storage can make devices slow.
  • Memory (RAM) is short-term working space. Many apps open at once can fill it up.
  • CPU is the “brain.” Background tasks can use it and slow everything down.
  • Updates often fix bugs and security issues, but may require a restart to finish.
